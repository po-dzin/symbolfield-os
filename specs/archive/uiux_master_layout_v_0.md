# SymbolField UI/UX Master Layout — v0.44

Полный концепт нового экранного макета: панели, доки, окна, режимы, кристальное погружение в ноду.

---

# 🧩 0. Примордиальный принцип

> **Граф — это мир. Панели — органы чувств. Время — вертикаль. Окна — инструменты. Нода — фрактальный кристалл.**

---

# 1. Основной каркас интерфейса

```
┌──────────────────────────────────────────────────────────────┐
│                     STATE PANEL (TOP)                        │
└──────────────────────────────────────────────────────────────┘
│ NAV BAR │                GRAPH CANVAS                │ TIME │
│  (L)    │             (2D / 2.5D / 3D)               │SPIRAL│
└──────────────────────────────────────────────────────────────┘
│                    WINDOWS DOCK (BOTTOM)                     │
└──────────────────────────────────────────────────────────────┘
```

## 1.1 Назначения

- **Nav (лево)** — входы, режимы, глобальные кнопки.
- **Graph (центр)** — рабочее пространство всей ОС.
- **Time Spiral (право)** — вертикальная временная ось.
- **State Panel (верх)** — состояния: Mode, Tone, Glyph, XP.
- **Windows Dock (низ)** — свёрнутые окна и быстрый доступ.

---

# 2. Nav Dock (левый)

- Фиксированная вертикаль.
- Кнопки 40px, отступы 2U.
- HUD / Graph / Agent / Log / Settings.
- Удержание → контекст мини-меню.

---

# 3. Graph Canvas (центр)

### 3 режима:

1. **Flat 2D** — чтение, структура, работа.
2. **2.5D Constellation** — параллакс, органика, созерцание.
3. **3D GraphMorph (будущее)** — полноценная пространственная сцена.

### Интеракции:

- drag → pan
- wheel/pinch → zoom
- dblclick → Crystal Dive Mode (погружение в ноду)

---

# 4. Time Spiral Dock (правый)

### Вертикальная временная ось:

- Уровни: day → week → month → year → 3y → 7y → 12y.
- Glow зависит от XP-насыщенности периода.
- Scroll → смена масштаба.
- Клик по сегменту → фильтрация графа.

Размеры:

- Ширина ≈ 0.236W.
- Мини-ячейка: 5U×8U.
- Ось: 1.5U.

---

# 5. State Panel (верхняя)

Содержит:

- Mode (🕳 deep / 🌀 flow / 🔆 luma)
- Tone (цвет)
- Glyph
- XP summary
- Harmonic Lock (ϕ)

Высота = \~0.146H. Анимации: 0.3–0.4s, glow-breath: 12–18s.

---

# 6. Windows Dock (нижний)

Как в ОС:

- Свёртывание любых окон.
- Иконки возврата.
- Double click → fullscreen overlay.
- Drag → reorder.

Размеры:

- Высота = 0.146H.
- Иконки = 5U.
- Превью окна: 13U×9U.

---

# 7. Crystal Node Dive Mode (Macrozoom)

### A. Погружение внутрь

- Дабл клик → камера входит в ноду.
- Граф растворяется в глубине.
- Внутри — кристаллический глиф-сущность.
- Параллакс пространства.

### B. Приближение ноды к нам

- Граф остаётся на фоне.
- Нода всплывает как сферокристалл.
- XP-ореолы вращаются.

Выбор в Settings: `approach / enter`.

### Содержимое Crystal-внутренности:

- XP-поля (кольца)
- Центральный глиф
- Связи как ветви
- Time-слои
- Agent-спутник (микро-медуза)

Панели Nav/State/Time остаются поверх.

---

# 8. Поведение окон

- Все окна: draggable + resizable.
- Minimize → System Dock.
- Restore → в исходную позицию.
- Fullscreen → overlay поверх графа.
